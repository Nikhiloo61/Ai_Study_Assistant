# Ai_Study_Assistant
AI Study Assistant is a smart web-based educational platform that combines Artificial Intelligence and web technologies to enhance the learning experience. The system is built using Flask for backend development and integrates a locally running LLaMA 3 model via Ollama to provide real-time chatbot functionality.
The application includes multiple features such as:
AI-powered chatbot for answering queries
Quiz module for knowledge assessment
Dashboard for tracking user performance
Secure login system for personalized access
One of the key highlights of this project is that it runs completely offline after setup, ensuring data privacy and independence from external APIs. This project demonstrates the practical implementation of Natural Language Processing (NLP) and Large Language Models (LLMs) in an interactive web application.
